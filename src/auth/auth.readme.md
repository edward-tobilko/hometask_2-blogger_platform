- Adapters -> все, что работает с «внешним миром» (шифрование, почта, токены).
- DTO / Command -> input от клиента.
- Repository -> достает обьект с БД.
- Domain -> работает только с валидными данными (описываем и работаем с бизнес логикой).
- Controller ( Handler ), Application -> презентационный слой.
